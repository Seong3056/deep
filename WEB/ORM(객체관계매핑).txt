Django ORM

ORM (Object-relational mapping,객체 관계 매핑)
- 데이터베이스의 데이터를 객체와 연결해주는 기능
- Model 클래스는 테이터베이스의 테이블의 형태를 나타냄
- Model클래스를 통해 테이터를 가져올 때는 모델의 object속성을 사용

- 전체 데이터 목록 가죠오기
-- 모델.object.all()

- 특정 조건을 만족하는 데이터 한개의 정보를 가져오기
-- 모델.object.get(조건)

- 특정 조건을 만족하는 데이터들의 정보 가져오기
-- 모델.object.filter(조건)
